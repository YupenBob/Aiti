# AITI - Silicon Type Indicator

类似 MBTI 的 16 种 AI 模型人格测试系统，帮助用户测定他们最符合哪个人工智能大模型。

---
Made by **York & Clover** · 2026

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
